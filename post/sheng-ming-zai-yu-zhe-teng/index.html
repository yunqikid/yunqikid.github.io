生命在于折腾

2021-03-05

想法初现

近几年见到博客还是在初升高的假期看了《神探夏洛克》中的华生因为战争时流下的阴影心理医生让他通过写博客把自己内心的想法发布到网上去缓解一下心情；再下一次就是去一个喜欢的视频博主的博客里看文章看到了许多平时他微博里没有的东西也学到了很多东西，突然觉得博客这种东西真的是太棒了，简直酷。拥有一个自己的博客可以

  • 记录的想法
  • 拥有一个属于自己的可编辑内容空间
  • 自定义程度极高

App Store里不乏像Notion、QQ空间、朋友圈、印象笔记、One note、熊掌记可以记录想法和生活甚至是系统备忘录，但他们无法随时随地的在不同的设备方便的查看，大多数的笔记软件只能在自己的账户里看到记录下的内容而博客基于网页http这一协议，保证了只要有网络就可以访问和浏览信息没有设备和账号和设备的限制。也有基于网页的社交平台比如CSDN，但是它有个致命的问题就是它的UI太绝了：

  • 界面给你安排的满满的，巨难看
  • 广告插入，影响观看
  • 看完整的评论还要注册账号
  • 手机端需要下载APP，手机网页端效果太糟糕
  • 手机网页端浏览效果太糟糕

一些网站里也有很好的内容，可就是它的界面真的很难让人长时间的去看去发掘好文章，在折腾电脑的时候很多解决方案是在简书和CSDN上找到的。

开始动手

刚开始的想法是想用Wordpress来搭建博客的因为没有想好域名选啥名字就一直没买服务器也没有去租，后来看见了用Hexo+GitHub Pages搭建博客关键是这个方案不需要去注册域名和租用服务器。第一次搭建博客就遇到了很多的坑比如：

  • Hexo初始化进度条卡住
  • 推送到Github失败
  • Local4000端口被占用
  • 重复的在同一个地方失败导致身心疲惫

后来在少数派有用Gridea来搭建博客，看完文章马上就动手开始搭建但是在用Gridea的时候也遇到了麻烦

  • Github的网站打开速度太慢了
  • 网站可以Ping通但连接不到服务器
  • 同步失败
  • 手机浏览器显示无法连接到服务器

GitHub Pages连接不上服务器通过修改DNS可以正常访问了；同步失败是因为分支选错了默认的分区是main分区不是master；直到用Gridea搭完博客还是不知道为什么Hexo在推到远端的时候一直卡住。现在电脑可以访问博客啦，但其他的设备还是不可以访问太糟糕了。还是需要一步步的折腾和学习。

来说说Gridea的优点

  • 极易上手
  • 写文章方便，支持markdown
  • 对新手及其友好

如果只是把博客拿去当做一个记录笔记的和思考的地方Gridea其实已经很够用了，问题主要是在GitHub Pages的访问问题。目前看来是因为DNS污染的问题导致其他设备不能访问，如果其他设备需要修改DNS才能访问也太麻烦了，要准备换一个托管平台了可能是是Wordpress也有可能是Hugo。

生命在于折腾嘛，什么事都嫌麻烦的话那睡觉是最好的选择了。
